# 📖 使用说明

## 快速开始

### 本地开发环境

1. **启动后端服务**（终端 1）
```bash
cd backend
npm install
npm start
```
后端将在 http://localhost:3001 运行

2. **启动前端服务**（终端 2）
```bash
cd frontend
npm install
npm run dev
```
前端将在 http://localhost:3000 运行

3. **访问应用**
打开浏览器访问 http://localhost:3000

## 功能说明

### 1️⃣ 创建邮箱

- 点击「立即创建邮箱」按钮
- 系统会自动生成一个随机邮箱地址和密码
- 邮箱格式：`随机字符串@mail.tm`

### 2️⃣ 查看邮箱信息

创建成功后会显示：
- 📧 完整邮箱地址
- 🔑 邮箱密码
- 🎨 一键复制按钮

### 3️⃣ 接收邮件

- 系统会自动加载收件箱
- 点击「刷新邮件」获取最新邮件
- 新邮件到达时会自动显示

### 4️⃣ 阅读邮件

- 点击邮件列表中的任意邮件
- 查看邮件详情：发件人、主题、内容
- 支持 HTML 格式邮件

### 5️⃣ 删除邮箱

- 使用完毕后点击「删除邮箱」
- 确认后即可彻底删除账户
- 保护隐私，不留痕迹

## 使用场景

✅ **适合使用的场景：**
- 注册临时账户
- 接收验证码
- 避免主邮箱收到垃圾邮件
- 测试邮件功能
- 保护个人隐私

❌ **不适合使用的场景：**
- 重要账户（银行、支付等）
- 需要长期使用的服务
- 存储重要邮件
- 找回密码（邮箱会被删除）

## 注意事项

1. **邮箱有效期**：建议用完即删，不要长期保留
2. **密码安全**：密码仅显示一次，请妥善保存
3. **隐私保护**：不要用于非法用途
4. **服务限制**：基于 Mail.tm 免费 API，有使用限制

## API 限制

Mail.tm 免费账户限制：
- 每个 IP 每小时创建邮箱数量有限制
- 邮箱有使用期限
- 不适合大量批量创建

## 故障排除

### 问题：创建邮箱失败
**解决方案：**
- 检查网络连接
- 确认 Mail.tm API 服务正常
- 稍后重试（可能触发频率限制）

### 问题：无法接收邮件
**解决方案：**
- 点击「刷新邮件」按钮
- 确认邮箱地址正确
- 检查垃圾邮件文件夹

### 问题：页面加载缓慢
**解决方案：**
- 检查网络连接
- 清除浏览器缓存
- 尝试使用其他浏览器
